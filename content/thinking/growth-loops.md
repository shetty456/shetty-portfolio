# Growth Loops: Sunil's Mental Model

## Why Loops Beat Funnels

Traditional growth thinking uses funnels: Awareness → Acquisition → Activation → Revenue → Retention. Funnels are useful but incomplete — they don't explain compounding.

Growth loops are systems where output from one cycle becomes input to the next. This creates compounding, not linear, growth.

## Types of Growth Loops

### Viral Loops
User action → Creates value for another person → That person joins

Example: Yelow's creator referral program. Creator earns money → Invites peer creator → Peer joins and creates content → Platform grows.

**Key metric**: Viral coefficient (K-factor). K > 1 = exponential growth.

### Content Loops
Users create content → Content attracts new users → New users create content

Example: YouTube, Twitter, TikTok. The content created by users IS the acquisition mechanism.

**Key metric**: Content → New user registration attribution.

### Product Loops
Users use product → Product improves (via data/network) → Better product attracts more users

Example: Spotify's Discover Weekly. More listening → Better recommendations → More listening → More data.

**Key metric**: Engagement depth over cohort lifetime.

## Applying Loops to PM Work
When evaluating a feature, Sunil asks: "Does this create a loop, or does it serve a one-time purpose?"

Features that create loops have compounding ROI. Features that don't are linear.
