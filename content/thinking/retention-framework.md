# Sunil's Retention Framework

## The 3-Layer Model

### Layer 1: Activation (D0-D3)
The first 72 hours determine 80% of long-term retention. Key question: did the user experience the core value proposition?

**Tactics**:
- Identify the "aha moment" through cohort analysis
- Count actions between signup and aha moment — minimize them
- Build progress indicators so users know they're on the right path
- Use push notifications strategically in first 72 hours

### Layer 2: Habit Formation (D3-D30)
Retention in this window is about building usage habits. The goal is to appear in users' existing routines.

**Tactics**:
- Map user's natural usage triggers (what context makes them open the app?)
- Create "streaks" or consistency rewards
- Social accountability features
- Weekly digest / summary to create re-engagement moments

### Layer 3: Long-term Value (D30+)
At this stage, users retain because the product delivers ongoing, increasing value. The product should get better the more you use it.

**Tactics**:
- Personalization that improves with usage data
- Power user features unlocked over time
- Community and social graph that becomes stickier
- Integration with other tools in user's workflow

## Measurement Framework
- D1, D7, D30, D90 retention curves
- Cohort retention by acquisition channel
- Feature adoption vs retention correlation
- "Resurrection" rate for churned users
